# 💰 Budget Buddy

A simple, private personal expense tracker. Log income and expenses by category, watch your running balance, and see a per-category spending breakdown. Everything stays in your browser.

**Live demo:** https://skrizidev.github.io/budget-buddy/

## Pages
- **Tracker** — balance/income/expense summary, add form, transaction history
- **Report** — spending by category with percentage bars
- **About** — privacy notes

## Tech
Vanilla HTML / CSS / JavaScript. Transactions in `localStorage`.
